# The Client-Side Toolbox

This repository hosts a set of client-side tools that run entirely in the browser. All processing happens locally, with no data sent to servers.

## Implemented Work
- A minimal JSON beautifier helper in script.js (and demo page for testing).
- Basic HTML scaffolding for future tools.

## How to Run
- Open index.html or the demo page directly in a browser. Use the browser’s Developer Console for testing the jsonBeautify function.
