About the Weather Dashboard

What is the Weather Dashboard?

The Weather Dashboard is a fast, privacy-first weather tool that fetches real-time data directly in your browser without any server or login. It shows current conditions, a 24-hour hourly strip, and a full 7-day forecast for any city in the world, or automatically for your current location.

Powered by Open-Meteo

Weather data is sourced from Open-Meteo, a free and open-source meteorological API that uses high-resolution numerical weather prediction models (ECMWF, GFS, and others). No API key is required, and Open-Meteo does not track or store personal data. Geocoding (converting city names to coordinates) is also handled by Open-Meteo's public geocoding endpoint.

Privacy & Client-Side Philosophy

In keeping with the client-side toolbox philosophy, this tool makes API calls directly from your browser to a third-party data provider. No data passes through any server owned by this toolbox. When you use My Location, your coordinates are requested via the browser's native Geolocation API and sent only to Open-Meteo — they are never stored, logged, or shared.
