About the Weather Dashboard

Four tools in one

The Weather Dashboard bundles four utilities: a live Forecast (current + hourly + 7-day), a Comparator for side-by-side city comparison, a Historical Lookup reaching back to 1940, and a Severe Weather risk timeline for the next 48 hours.

Powered by Open-Meteo

All data is sourced from Open-Meteo, a free and open-source meteorological API using ECMWF, GFS, and ERA5 models. No API key required. City geocoding also uses Open-Meteo's free geocoding endpoint.

Privacy & Client-Side

API calls go directly from your browser to Open-Meteo — no data touches any server owned by this toolbox. When you use My Location, your coordinates are sent only to Open-Meteo and are never stored or logged here.
